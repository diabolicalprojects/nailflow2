
# Deployment Model

Each nail business receives its own deployment.

## Deployment Steps
1 Clone repository
2 Configure environment variables
3 Configure business profile
4 Connect n8n webhook
5 Deploy

## Environment Variables

DATABASE_URL
N8N_WEBHOOK_URL
CDN_ENDPOINT
MERCADOPAGO_API_KEY
PAYMENT_TEST_MODE
BUSINESS_NAME
BRAND_COLOR_PRIMARY
BRAND_COLOR_SECONDARY
