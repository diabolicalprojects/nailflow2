
# NailFlow UI Redesign Prompt for Development Agent

## Objective
Replace the current UI/UX generated by the agent with a new design system and workflow based on the provided mobile-first mockups. The new design must strictly follow the screen flow and visual style shown in the reference images.

The system is already built using:
- Next.js (frontend)
- Node.js + Express (backend)
- PostgreSQL
- Responsive layout

You must **refactor only the UI/UX layer**, without breaking backend logic, API routes, or booking/payment functionality.

---

# Design Philosophy

The new design must follow these principles:

- Mobile‑first interface
- Soft pastel aesthetic
- Feminine and minimal visual language
- Rounded components
- Clean spacing
- Smooth card‑based layout
- Soft shadows
- Pastel pink primary theme

# Responsive Rules

The layout must remain **mobile‑style even on desktop**.

Desktop should NOT convert to a traditional dashboard layout.

Instead:

Desktop view must center a **mobile container UI**.

Example:

| Desktop Screen |
|                |
|     Mobile UI  |
|                |

Max width of UI container:
480px – 520px

Centered horizontally.
